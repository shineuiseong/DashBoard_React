## 🌻 소개

- Material UI를 사용하여 대시보드 UI 초기 구성

- 다른 프로젝트 진행시 관리자모드로 활용 가능

- 대시보드 UI를 활용하기위해서 구성한 프로젝트이므로 로그인, 회원가입, 설정 등은
  실제 프로젝트 적용시 구현예정

- 임의로 설정한 필드값이기 때문에 차후 node 백엔드와 통합하여 데이터구성 필요
